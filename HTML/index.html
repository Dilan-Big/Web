Propiedades

Las propiedades CSS son instrucciones que definen el estilo, layout y comportamiento de elementos HTML, agrupadas en categorías como Box Model, tipografía o Flexbox. Se aplican vía selectores y son esenciales para responsive design en tu bootcamp y ecommerce.

Box model

El Box Model en CSS define la estructura rectangular de cada elemento HTML como una caja con cuatro capas: contenido, relleno (padding), borde (border) y margen (margin). Es la base del layout y explica por qué el tamaño total no siempre coincide con width + height.

Selectores

Un selector en HTML se refiere principalmente a los selectores CSS, que se usan para apuntar a elementos específicos del código HTML y aplicarles estilos. No forman parte del HTML puro, sino que se escriben en hojas de estilo CSS para identificar etiquetas, clases, IDs u otros atributos.

Display

La propiedad display en CSS controla cómo se muestra un elemento HTML en la página, definiendo su comportamiento de layout como bloque, en línea o contenedor flexible. Se usa con selectores para cambiar el flujo visual, esencial en diseños responsive para tu tienda de ecommerce

FloxBox

Flexbox es un módulo de CSS que crea layouts unidimensionales flexibles, distribuyendo y alineando elementos hijos en un contenedor padre de forma responsive y eficiente. Se activa con display: flex y es ideal para menús, galerías o cards de productos en tu ecommerce de dropshipping.

Medidas

Las unidades de medida (o medidas) en CSS definen tamaños como ancho, alto, márgenes o fuentes, dividiéndose en absolutas (fijas) y relativas (adaptables). Son clave para diseños responsive en tu ecommerce, combinándolas con Flexbox y display.
